10000=Load Time

Max of each bait to attempt to buy each time you return to the vendor. If none of the locations you have selected to fish below contain this bait, it will not be bought. To fish a given spot, set the location's value to 1 or greater. 0 means you do not want to fish there at all. 1 will be fished first, 2 second, etc. Multiple fishing spots can have the same number, and when they do, they will be
fished in alphabetical order. So you can think of this number as a "rank". Once all of rank 1 are complete, it will move on to rank 2, and so on. Once it goes through all ranks, it will begin rank 1 again.

For the bait itself, the only valid values are 0 or 1. A value of 0 means you don't want to use that bait, while a 1 means that you do. Bait will be used from top to bottom; there is no ranking system like the previous.
